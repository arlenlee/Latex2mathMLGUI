# Latex2mathMLGUI
需要环境 python + re + tkinter + pyperclip +latex2mathml

使用看intro
复制latex公式后以$或$$开始的文本点击'covert'自动转换为mathML可直接粘贴到Word里面
呈现出一个公式 而不是latex公式源码
